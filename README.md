# twitter-like-shell

- twitterみたいな対話形式のシェル（っぽいただのデスクトップアプリケーション）

## 使用技術

- Typescript
- Node.js
  - Electron
  - React

## 始め方

1. install node.js
2. install npm
3. clone this repository
4. $ npm ci
5. $ npm run dev
